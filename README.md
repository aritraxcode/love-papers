# Love Papers ‚ù¶

**Love Papers** is a romantic, vintage-styled web application that allows users to create beautiful, newspaper-style love letters and declarations. Designed with a timeless aesthetic, it lets you craft personalized messages, upload photos with automatic sepia filters, and download your creation as a high-quality image.

## ‚ú® Features

- **Vintage Themes**: Choose between multiple newspaper styles (e.g., "Love Gazette", "Daily Valentine").
- **Visual Editor**: Real-time preview of your love letter as you type.
- **Image Processing**: Automatically applies vintage/sepia filters to uploaded photos to match the aesthetic.
- **Mobile Responsive**: Fully functional on mobile devices, tablets, and desktops.
- **High-Quality Download**: Export your letter as a high-resolution PNG, perfect for sharing or printing.
- **Date Picker**: Easy date selection with a custom integrated picker.

## üöÄ Getting Started

Follow these steps to set up the project locally.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/aritraxcode/love-papers.git
    cd love-papers
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Navigate to `http://localhost:5173` (or the port shown in your terminal).

## üì¶ Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` directory.

## üìù Usage

1.  **Enter Details**: Fill in your name, your partner's name, and a heartfelt message.
2.  **Pick a Date**: Select a special date (anniversary, Valentine's Day, etc.).
3.  **Upload Photo**: Add a photo of you or your partner. It will automatically be styled.
4.  **Choose Theme**: Toggle between different newspaper styles using the theme switcher.
5.  **Download**: Click the "Download Love Letter" button to save your creation.
